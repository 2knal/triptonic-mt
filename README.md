### TripTonic - REST API Backend

---

- Add .env file to the root

```
GOOGLE_GEMINI_API_KEY=
GOOGLE_MAPS_API_KEY=
MONGO_CONNECTION_STRING=
WEBSITE_DOMAIN=
```

- Install dependencies by running
  `pip install -r requirements.txt`

- Run server as follows
  `python app.py`
